# Canvas Case Roller

This program allows you to open a case everytime you submit one of your canvas assignmnets. Once you complete an assignment click on it and see your case roll to find out what item you got. Collect as many rare items as you can.


## Instructions for Build and Use

Steps to build and/or run the software:

1. Open Qt creator
2. Run software

Instructions for using the software:

1. Write your canvas LMS api token in the code
2. Run the software and wait for the program to load all of your assignments
3. Click a completed assignment to roll a case
4. View the items you receive by clicking the view item button

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Qt Creator 18.0.0 (License for students)
* C++
